# FoodieHub — Frontend-only Food Ordering UI

This is a responsive, accessible, frontend-only food ordering website built with HTML, CSS, and JavaScript. All data is simulated in `assets/js/app.js` and the cart/theme state is stored in `localStorage`.

Files:
- index.html — Home
- menu.html — Menu (search, filter, sort)
- food-details.html — Details for an item
- cart.html — Shopping cart
- order-summary.html — Order breakdown
- checkout.html — Checkout form (simulated)
- about.html — About page
- contact.html — Contact form
- assets/css/style.css — Main styles
- assets/js/app.js — Data and page logic
- assets/images — SVG placeholders

To run: open `index.html` in a browser.

No backend required.
